# 玄机 · AI 易学决策助手

八字排盘 · 奇门遁甲 · 梅花易数 — 三合一 AI 辅助决策工具

## 八字排盘引擎（MVP）

纯前端 JavaScript 实现，零后端零 API 成本。

### 功能

- ✅ 四柱八字（年/月/日/时柱，考虑立春节气分界）
- ✅ 十神计算（基于五行生克环）
- ✅ 地支藏干及十神
- ✅ 五行统计（可视化柱状图）
- ✅ 大运排盘（8步大运，当前运高亮）
- ✅ 纳音
- ✅ 格局分析（五行平衡、午午自刑、十神组合）
- ✅ 暗黑玄学风 UI
